# Full Stack E-commerce-Website
A full-stack e-commerce web application that simulates real-world product ordering, shipment tracking, and order management. Built using Node.js, Express.js, JavaScript, and SQL.

## Features
- User account creation and login
- Browse products and view product details
- Add/remove items from cart
- Real-time order creation and cancellation
- Shipment tracking with dynamic order updates
- Update shipping speed and delivery address
- SQL-backed order management system

## Tech Stack
### Frontend:
- HTML
- CSS
- Javascript (Fetch API, async/await)

### Backend:
- Node.js
- Express.js
- RESTful API design

### Database:
- SQL

## System Architecture
- RESTful API endpoints handle client-server communication
- Backend processes business logic and interacts with SQL database
- Database stores users, products, orders, and shipment details
- Asynchronous operations improve responsiveness and user experience
